# Express TypeScript Template

Look at the [Express Documentation](https://expressjs.com/en/starter/installing.html) to learn more.

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## Development Server

Start docker containers:

```bash
docker compose up -d
```

Start the development server on `http://localhost:3000`, or you can change port in `.env`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```


## Production

Build the application for production:

```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build
```
